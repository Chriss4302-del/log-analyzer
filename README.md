# Log Analyzer

This project is a simple Python script that reads a log file and analyzes login activity.

The goal was to get more hands-on experience with logs and understand how login attempts can be identified and summarized.

## What it does

- Reads a log file
- Counts failed login attempts
- Counts successful logins
- Prints a simple summary

## Why I made this

After working with logs in my cloud VM project, I wanted to explore them a bit more and see how they could be analyzed using Python.
